* HashSet 简介

    * HashSet 是一个没有重复元素的集合。
    * HashSet 由HashMap实现的，不保证元素的顺序，而且HashSet允许使用 null 元素。
    * HashSet是非同步的。
    
* HashSet的主要API
    ```
    boolean         add(E object)
    void            clear()
    Object          clone()
    boolean         contains(Object object)
    boolean         isEmpty()
    Iterator<E>     iterator()
    boolean         remove(Object object)
    int             size()
    ```
